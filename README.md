# photo-ops
An automation pipeline following a microservice architecture to crop, caption, hashtagify and post photos to social media.

## Dependencies

- [Imaginary](https://github.com/h2non/imaginary)
  - Responsibilities:
    - Image cropping
    - Image resizing
    - Image compression

## Prerequisites

### Running the application

- docker
- docker-compose

### Development

- docker
- docker-compose
- Go
- bash
